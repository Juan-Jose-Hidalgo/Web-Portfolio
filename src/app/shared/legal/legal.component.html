Finalidad del tratamiento de datos personales:

Los datos personales proporcionados por los usuarios en el formulario de contacto de este sitio web se recopilan con el fin de permitir que el titular del sitio web reciba y responda a las consultas y solicitudes enviadas a través del formulario.

Legitimidad del tratamiento de datos personales:

El tratamiento de los datos personales se basa en el consentimiento explícito y voluntario del usuario proporcionado a través del formulario de contacto.

Destinatarios de los datos personales:

Los datos personales no serán comunicados ni cedidos a terceros, salvo en los casos en que exista una obligación legal de hacerlo.

Plazo de conservación de los datos personales:

Los datos personales proporcionados por los usuarios se conservarán únicamente durante el tiempo necesario para cumplir con la finalidad para la que se recopilaron, a menos que el usuario solicite su supresión.

Derechos del usuario:

El usuario tiene derecho a acceder, rectificar, suprimir, oponerse, limitar o portar sus datos personales. Para ejercer estos derechos, el usuario puede ponerse en contacto con el titular del sitio web a través del correo electrónico de contacto proporcionado en este aviso legal.

Cambios en el aviso legal:

El titular del sitio web se reserva el derecho de modificar este aviso legal en cualquier momento, siempre que lo considere necesario o apropiado. Las modificaciones se publicarán en el sitio web y se entenderán aceptadas cuando el usuario continúe utilizando el sitio web después de su publicación.

Con la aceptación de este aviso legal, el usuario declara haber leído y comprendido la información proporcionada sobre el tratamiento de sus datos personales por parte del titular del sitio web.
